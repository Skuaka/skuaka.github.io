LINUX 小命令记录

A

awk：文本查找

另见

B

C

crontab

定时执行任务

D

du

查看文件夹占用空间

E

F

G

grep：文本查找

  • -v 反向查找
  • -r 递归查找文件夹内符合的文件

H

hexdump

十六进制查看文件

history

查看命令行记录

  • -c 清除命令行记录

直接清除历史记录文件：echo > ~/.bash_history

host

查看主机名/网站 ip

I

iftop

和top类似，不过显示的是网络数据收发。

locate

和 find 功能类似，但速度快，因为有定时更新的系统路径数据库。

J

K

L

last

查看所有的登入信息

  • -n 只看最近 n 条
  • -R 不显示 ip/hostname
  • username 只看指定用户

lastlog

查看所有用户最后一次的登入时间

lsof

lsof -p PID

查看进程打开的文件

lsof -i

查看联网进程

M

N

nslookup：网站ip

查看网站ip 也可以用 host 命令

nmap：内网扫描

另见

nc

NetCat

被设计为一个简单、可靠的网络工具，可通过TCP或UDP协议传输读写数据。同时，它还是一个网络应用Debug分析器，因为它可以根据需要创建各种不同类型的网络连接。

监听http请求内容：

O

P

Q

R

S

sftp

ls 查看当前目录下文件

help 查看sftp支持哪些命令

cd 指定目录

pwd 查看当前目录

get xxx.txt 下载xxx文件

put xxx.txt 上传xxx文件

quit / bye / exit 退出sftp

sed

见另一篇

strace

strace -p PID

查看该进程调用的所有系统调用

shutdown

shutdown [minutes]

在minutes分钟后关机

T

U

uname

uname -a

查看操作系统信息

V

W

w

查看当前登入的用户

who

查看当前登入的用户

who /var/log/wtmp 显示从wtmp文件创建或删改以来的每一次登录

X

xargs

xargs是给命令传递参数的一个过滤器，也是组合多个命令的一个工具。

https://www.runoob.com/linux/linux-comm-xargs.html

统计文件夹内所有.h文件行数 find . -name "*.h"|xargs cat|wc -l
